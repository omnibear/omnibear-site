# Contributing to the Omnibear Website

Welcome to the Omnibear website repository.
We welcome contributions to this project.

## Getting Started

1. Install hugo ([install instructions](https://gohugo.io/installation/))
1. Run the dev server `hugo server` to launch a local dev server showing the site
1. Check out the [Help Wanted](https://github.com/omnibear/omnibear-site/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22help%20wanted%22) tag in the GitHub issues for ways to contribute.

## PR Submission Guidelines

1. Avoid large PRs without previous discussion in issues
1. Follow the existing coding standards
1. Ensure the site builds
1. Clearly describe what the change does in the PR title/description
